# 🍱 MealMind

An AI-powered Food Prep app built with Flutter that helps you plan, discover, and save meals — personalized to your taste.

---

## ✨ Features

- **AI Home Recommendations** — Daily dish recommendations generated by Gemini AI based on your saved preferences. Regenerates once per day unless manually refreshed.
- **Meal Planner** — Generate a 1–7 day meal plan using AI. Plan is saved locally and persists across app restarts.
- **AI Assistant (MealMind Chef)** — Chat-based dish discovery. Ask for any dish and get a full recipe card with ingredients, steps, and nutrition. Strictly limited to food topics.
- **Favorites** — Save any recipe to your favorites list. Real-time updates across the app without needing to restart.
- **Recipe Detail** — Full recipe view with ingredients, step-by-step instructions, and nutrition facts. Add to favorites directly from the detail page.
- **Refine Preferences** — Customize your food themes, halal toggle, and additional notes. Saved via Hive and used to personalize AI prompts.

## 🏗️ Architecture

Clean Architecture (Uncle Bob) with the following layers:

```
lib/
├── core/
│   ├── database/        # Hive local storage (AppDatabase, FavoritesLocalDataSource)
│   ├── mappers/         # Entity → UI model mappers
│   ├── theme/           # AppColors, AppTextStyles, AppTheme
│   └── widgets/         # Reusable widgets (AppButton, AppChip, etc.)
│
└── features/
    ├── home/
    │   ├── data/        # RemoteDataSource, LocalDataSource, Prompts, Models
    │   ├── domain/      # Entities, Repository interface, Use cases
    │   └── presentation/ # BLoC, Pages, Widgets
    ├── planner/
    │   ├── data/        # RemoteDataSource, LocalDataSource, Prompts
    │   ├── domain/      # Entities, Preference model
    │   └── presentation/ # BLoC, Pages, Widgets
    ├── ai_assistant/
    │   ├── data/        # DataSource, Models, Prompts
    │   └── presentation/ # Page, Widgets
    ├── favorites/
    │   └── presentation/ # Page, Widgets
    └── recipe_detail/
        └── presentation/ # Page, Widgets
```

**State Management:** Flutter BLoC  
**Local Storage:** Hive  
**AI:** Firebase AI Logic (Gemini 2.5 Flash)  
**Backend:** Firebase

## 🚀 Getting Started

### Prerequisites

- [FVM](https://fvm.app/) installed
- Flutter SDK managed via FVM
- Firebase project configured (see `firebase_options.dart`)

### Run the app

```bash
fvm flutter pub get
fvm flutter run
```

### Regenerate app icons (after changing `asset/icons/app_icon.png`)

```bash
fvm dart run flutter_launcher_icons
```
